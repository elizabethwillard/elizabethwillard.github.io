## Key Concepts

### Vectors
- A vector is an ordered collection of numbers
- Can be represented as a column or row
- Operations: addition, scalar multiplication, dot product

### Linear Transformations
- A function that maps vectors to vectors
- Preserves vector addition and scalar multiplication
- Can be represented as a matrix

## Examples

1. **2D Rotation**: Rotating a vector in 2D space by angle θ
2. **Scaling**: Multiplying all components by a scalar

## Key Takeaways

- Vectors form the foundation of linear algebra
- Linear transformations are the main focus of the subject
- Matrices are the computational tool for linear transformations
