Resources:

The Practice of Cloud System Administration Volume 2: Designing and Operating Large Distributed Systems by Thomas A. Limoncelli, Strata R. Chalup, and Christina J. Hogan (O'Reilly Media, 2024) - This book provides insights into designing and operating large-scale cloud systems, making it a valuable resource for AWS certification preparation.
There are no tutorials like "How to set up an EC2 instance" in this book, but it covers the principles and thinking one must adopt to successfully implement distributed systems while also addressing fundamentals such as CAP Theorem, ACID, Sharding, and caching.
